## Guia de Operação

Utilize o Monitor Serial (9600 baud) e os seguintes comandos:

* **U** → Move o braço para cima
* **D** → Move o braço para baixo
* **O** → Abre a garra
* **C** → Fecha a garra

## Especificações Técnicas

* Arduino Uno
* 2 Servomotores (pinos 9 e 10)
* LED de status no pino 7
* Alimentação: 5V
